Product Description

Rotate is a desktop manipulator that will randomly flip the desktop into all sorts of weird positions. It can flip it over, reverse it or flip and reverse it. For an added benefit we change the users cursor to a "?" to aid in the confusion.

Each time you run Rotate it is different, so keep playing.

You can also view the product information online for Rotate.
