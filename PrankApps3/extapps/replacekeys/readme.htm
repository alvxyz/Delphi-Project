The ReplaceKeys prank replaces key presses with other keyboard keys.

Product Description

ReplaceKeys is a fun little prank that allows you to replace many keyboard keystrokes with different keyboard keystrokes. Let us explain....First you set the key you want to replace, in our case we will use the letter A. Next you set the key you want to replace A with, in our case we will use Z. Lastly you set how often it should replace the key press, in our case we will use 30 seconds. Then you start the prank! This is what will happen: after 30 seconds has passed ReplaceKeys will wait for the letter A to be pressed on the keyboard, once it has it will grab the A keystroke and send the letter Z to the computer. It works as if the person typed in the Z character!

ReplaceKeys makes it appear that the user has mis-typed on the keyboard. It will take them forever to realize you pranked them. ReplaceKeys is completely hidden and can be stopped easily by moving the mouse cursor to the TOP LEFT corner of the screen. ReplaceKeys is so effective we even played it on our president and he was fooled!

ReplaceKeys works with any program you are using within Windows. Yes, that means the web browser, email, Microsoft Office, etc. The possibilities are endless with this little prank.

How to Close

Move your mouse pointer to the TOP LEFT corner of your screen. This will stop the program from running in memory. If you have successfully moved your mouse in the correct position, the program will close and display an About Box Screen. Once you close this screen, the program will exit. Exiting the software will allow Windows to delete the file.

Command Line Options

replacekeys.exe [key to replace] [replace with key] [how often]

[key to replace] is the character or keyname of the key you wish to replace
[replace with key] is the character or keyname of the key you want to replace it with
[how often] is a number that specifies how many seconds you want to replace the keystroke. Use 0 for always.

"C:\replace keys\replacekeys.exe" a TAB 0

This would replace the "a" keypress with the TAB keypress all of the time

"C:\replace keys\replacekeys.exe" s a 45

This would replace the "s" keypress with the "a" keypress no less than every 45 seconds
